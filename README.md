# delivery-management-dashboard
Project delivery dashboard with RAID log, sprint tracking, milestone monitoring, capacity planning and project health reporting.

Managing multiple deliverables across a project requires visibility into sprint progress, risks, blockers, milestones, and team capacity. This project demonstrates a practical delivery dashboard built in Excel that enables project managers to monitor project health using standardized delivery metrics.
**Features:**
Sprint Velocity Tracking

RAID Log

Milestone Dashboard

Team Capacity Planning

Risk Register

Dependency Tracking

Delivery Health Monitoring

Executive Reporting

Google Sheets
